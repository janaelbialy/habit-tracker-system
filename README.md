# Habit Tracker System — Build Habits. Break Limits.

A simple and user-friendly web application designed to help users build, track, and maintain daily habits through consistency, motivation, and visual progress tracking.

---

## 🚀 Overview

Building habits is difficult due to lack of consistency and clear feedback.  
This system solves that by offering a simple daily tracking experience with streak motivation and visual analytics.

> One click per day is all it takes to stay consistent.

---

## ✨ Features

- 🔐 User authentication (secure login & registration)
- 🗂️ Create, edit, and delete custom habits
- ✅ One-click daily check-in system
- 🔥 Streak tracking (current & best streaks)
- 📊 30-day progress analytics
- 📈 Visual charts and performance tracking
- 📱 Fully responsive design
- 🎨 Clean UI with earthy theme (#38512F)

---

## 🖥️ Pages

- **Login / Register** → simple authentication system  
- **Dashboard** → today’s habits and quick tracking  
- **Statistics** → habit performance and analytics  

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript  
- **Backend:** Node.js + Express (or PHP/Laravel)  
- **Database:** MySQL  
- **Security:** Password hashing, sessions, input validation  

---

## 🗃️ Database Structure

- **Users** → account information  
- **Habits** → user-created habits  
- **DailyLogs** → daily completion tracking  
- **Analytics** → streaks and statistics  

---

## ⚙️ How It Works

1. User registers and logs in  
2. User creates daily habits  
3. User marks habits as “Done” each day  
4. System updates streaks and analytics  
5. User views progress through dashboard insights  

---

## 🎯 Goal

To help users transform small daily actions into long-term success through consistency, tracking, and motivation.
